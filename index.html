Install Scrub It (Beta)

Scrub It is currently in Beta and not yet on the official Chrome Web Store. Follow these 4 easy steps to install it manually.

  1. Download the extension package. Download .zip
  2. Unzip the file you just downloaded.
  3. Go to chrome://extensions in your Chrome browser and toggle on Developer mode (top right).
  4. Click Load unpacked (top left) and select the unzipped scrub_it_v1 folder.
Tip: Pin the extension to your toolbar for easy access!
